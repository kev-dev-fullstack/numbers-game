# Programmeringsuppgift – Svenska Spel – Senior Frontend (Team Nummer)

## Kom igång

```bash
npm install
npm run dev
```

Projektet körs sedan på [http://localhost:3000](http://localhost:3000).

---

## Mina anteckningar

### Vad gick bra?

Nöjd med strukturen på projektet och att jag fick lära mig mer om nextJS och tailwind. Jag försöker hålla UI och logik separerat och hålla det modulärt med kataloger.

### Vad gick mindre bra?

UI-designen kan bli bättre och det saknas även tester.

### Vad var den största utmaningen?

Shuffle och randomize, fick kolla upp de delarna. Har använt AI för dessa.
Namngivning brukar jag också fundera mycket kring, vad är läsbart för andra och inte bara för mig?

### Vad skulle du göra om du hade mer tid?

Jag hade lagt till tester och förbättrat UI.

### Skulle du gjort något annorlunda om du fick börja om?

Jag skulle lägga mer fokus på tester och tillgänglighet. Enhetstester för delar av logiken (shuffle-funktionen och generering av den korrekta raden) hade varit bra för att säkerställa förväntat beteende. Tillgänglighet hade kunnat förbättras med relevanta attribut som tex aria på rätt ställen.
